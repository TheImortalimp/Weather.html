# OpenWeatherMap Browser App

Browser-first weather web app using open data APIs and explicit user-permission flows.

## Features

- Local digital clock with automatic timezone detection
- Device compass heading (when browser/device supports it)
- Current weather data (Open-Meteo)
- Map display and marker (Leaflet + OpenStreetMap)
- Optional nearby places and place photos with graceful fallback behavior
- Live refresh every 2.5 minutes

## File

- `openweathermap.html`

## Browser Usage

1. Host this app on HTTPS using GitHub Pages or another static host.
2. Open it in any modern browser.
3. Grant location permission to use GPS weather + local establishments.
4. Enable compass in the page when heading support is available on device/browser.

## Access Requirement

- End users only need an internet connection and a modern browser.
- No APK packaging or app-store publishing is required.
